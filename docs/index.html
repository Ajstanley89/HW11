Summary: Know Your Music

What is more popular: Country or Rap? We tried to answer that question using data! We took all the top songs from the Billboard Hot-100, Country, and Rap charts, and we got audio analysis on all of them using the Spotify API.

Our analysis showed that Rap was more popular than country music, but why might that be? Checkout the in-depth analysis we performed in the Data sectioin to find out more.
